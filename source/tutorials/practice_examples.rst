Practice Examples
=================

The companion repository `apexrl_example <https://github.com/Atticlmr/apexrl_example>`_
contains runnable practice examples that connect ApexRL to Genesis and Atari
tasks.

Available examples
------------------

- ``go2_example``: Genesis Unitree Go2 locomotion with PPO and SAC scripts.
- ``drone_example``: Genesis Crazyflie hovering with PPO.
- ``breakout_dqn_example``: Atari Breakout with DQN.

Clone the examples repository next to ApexRL:

.. code-block:: bash

   git clone https://github.com/Atticlmr/apexrl_example.git
   cd apexrl_example

Install ApexRL and the environment packages into your active Python
environment, then follow the commands in the example repository README.
