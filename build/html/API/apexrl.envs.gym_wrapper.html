apexrl.envs.gym_wrapper module¶

Gymnasium environment wrapper for VecEnv interface.

This module provides wrappers to use Gymnasium environments with the vectorized environment interface used by ApexRL algorithms.

class apexrl.envs.gym_wrapper.GymVecEnv(env_fns, device='cpu')[source]¶

Bases: VecEnv

Vectorized wrapper for Gymnasium environments.

Wraps multiple Gymnasium environments to provide a VecEnv interface. Each environment runs independently (CPU-based).

Example

>>> from apexrl.envs.gym_wrapper import GymVecEnv
>>>
>>> def make_env():
...     return gym.make("CartPole-v1")
...
>>> env = GymVecEnv([make_env for _ in range(8)])
>>> obs = env.reset()
>>> actions = torch.randint(0, 2, (8,))
>>> next_obs, rewards, dones, extras = env.step(actions)
