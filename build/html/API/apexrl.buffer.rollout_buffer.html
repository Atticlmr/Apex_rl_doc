apexrl.buffer.rollout_buffer module¶

Rollout buffer for PPO algorithm.

Supports multi-dimensional observations (e.g., images) and flexible storage.

class apexrl.buffer.rollout_buffer.RolloutBuffer(num_envs, num_steps, obs_shape, device, num_privileged_obs=0)[source]¶

Bases: object

Buffer for storing rollout data during PPO training.

Stores transitions (observations, actions, rewards, dones, values, log_probs) and computes advantages using Generalized Advantage Estimation (GAE).

Supports multi-dimensional observations (images, vectors, etc.).
